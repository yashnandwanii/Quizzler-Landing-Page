Key Data Flows

Authentication Flow

User signs in via Google OAuth ‚Üí Firebase Auth validates credentials ‚Üí User profile is created/fetched from Firestore ‚Üí Session stored in GetStorage for persistence ‚Üí App navigates to Home Screen.

Quiz Flow

User selects category ‚Üí Quiz API Service fetches questions from quizapi.io ‚Üí Questions displayed with timer ‚Üí Answers validated ‚Üí Score calculated ‚Üí Results saved to Firestore ‚Üí Leaderboard updated.

AI Revision Flow

Wrong answers collected from quiz history ‚Üí Gemini AI generates personalized revision questions ‚Üí Custom quiz created with adaptive difficulty ‚Üí Progress tracked for improvement analytics.

Data Synchronization

Coins cached locally with 30-second validity ‚Üí Force refresh from Firebase when needed ‚Üí Leaderboard ranks recalculated on score update ‚Üí Achievements checked after each quiz completion.
